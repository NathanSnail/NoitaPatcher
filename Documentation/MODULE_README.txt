The noitapatcher.lua file contains signatures for the NoitaPatcher functions.

These use the EmmyLua syntax which certain tools can use for code completion/validation.

- JetBrains Integration: https://emmylua.github.io/installation/repo.html
- VSCode Integration: https://marketplace.visualstudio.com/items?itemName=sumneko.lua
    - Add this directory to the 'Lua > Workspace: Library' settings

- About EmmyLua https://emmylua.github.io/index.html
